# 01-Set-Exercises
## Challenge: User Access Dataset Analysis

1. Create two sets, set_a and set_b, representing user IDs collected from two independent services.
2. Determine the complete collection of unique user IDs across both datasets.
3. Identify the user IDs shared by both services.
4. Print the resulting sets along with the total number of unique users and shared users.
5. Use only Python's built-in set methods and operators to complete the analysis.
