# 01-Libraries
## Challenge:  Horror Movie Ratings Visualization

1. Import the necessary libraries, pandas and matplotlib.pyplot, to handle data manipulation and plotting.
2. Initialize a dictionary containing the curated names of 15 top-tier horror movies of 2026 alongside their predicted or actual ratings.
3. Convert the movie data into a structured Pandas DataFrame to enable efficient data cleaning and sorting.
4. Sort the DataFrame by ratings in descending order (highest to lowest)
5. Display the fully formatted and sorted dataset table in the console with clear division lines.
6. Design and display a clean vertical bar chart using Matplotlib, keeping the canvas dimensions to 10x5 inches.
7. Rotate the X-axis labels by 45 degrees for peak readability, and focus the Y-axis range between 5 and 9 to highlight the rating differences.
