# 01-Conditional
## Challenge: Grade Analyzer & Validator

1. Create a program that receives a student's score from the user (Supports decimal numbers/Float).
2. Implement a robust control flow using if, elif, and else statements to evaluate the score based on these criteria:
- Exactly 20 = Display a top-tier success message.
- Between 17 and 19.99 = Grade A equivalent message.
- Between 14 and 16.9 = Grade B equivalent message.
- Between 10 and 13.9 = Pass with improvement warning.
- Exactly 10 = "Red line" pass warning.
- Below 10 = Fail message.
3. Include an error-handling branch (else) for invalid inputs (e.g., scores above 20 or negative numbers).
4. Enhance the console user experience using clean formatting and visual separators.
